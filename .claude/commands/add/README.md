# Add Commands

Commands for adding features to existing WinForms applications.

## Available Commands

- **/add/data-binding** - Setup data binding for WinForms controls
- **/add/error-handling** - Add comprehensive error handling to code
- **/add/logging** - Setup comprehensive logging with Serilog or NLog
- **/add/settings** - Setup application configuration and user settings
- **/add/validation** - Add comprehensive input validation to forms
- **/add/test** - Add test coverage for existing code

## Usage

Use these commands to enhance existing forms and components with additional functionality.

All additions follow:
- WinForms best practices
- Proper separation of concerns
- Async/await patterns
- Error handling standards
- Security guidelines
