# Create Commands

Commands for creating new WinForms components.

## Available Commands

- **/create/form** - Create a new Form with MVP pattern
- **/create/dialog** - Create a dialog form with proper result handling
- **/create/service** - Create a new service class with dependency injection
- **/create/repository** - Create a repository class with Entity Framework Core
- **/create/custom-control** - Create a reusable custom UserControl

## Usage

Use these commands to scaffold new components following WinForms best practices and patterns.

All generated code follows:
- MVP/MVVM patterns
- Async/await for I/O operations
- Proper resource management
- Input validation
- Error handling with logging
- XML documentation
