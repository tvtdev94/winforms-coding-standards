# Refactor Commands

Commands for refactoring existing WinForms code to follow best practices.

## Available Commands

- **/refactor/to-mvp** - Refactor existing WinForms code to MVP pattern

## Usage

Use these commands to improve the architecture and structure of existing code.

Refactoring benefits:
- Better testability
- Improved maintainability
- Clearer separation of concerns
- Easier to extend and modify
- Follows industry standards
